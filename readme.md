## 使用方法
<p>AppleOSDownloader [/f][destion_folder]</p>
<li>当前路径下载</li>
<pre>
./AppleOSDownloader
</pre>

<li>指定路径下载</li>
<pre>
./AppleOSDownloader /f /Users/vkenchen/Sources/apple/source
</pre>
